Sources

Three libraries, one chooser

Each source is its own library with its own artwork, stats and hub. They stack into a chooser — and Beam reopens the one you used last.

Mega. Sign in once; your cloud becomes a poster wall. An embedded engine serves files directly — nothing downloads first.
Seedbox (HTTPS). Point Beam at any HTTPS file server for a second cloud library with its own isolated engine — a misbehaving box never touches Mega playback.
Local drives. USB and internal storage with per-drive Movies/TV assignment. Incremental scans; detached drives are remembered and flagged, not forgotten.

Playback

Playback that respects the source

Original bytes to the decoder, tuned to hold on hardware as old as the 2015 NVIDIA SHIELD.

Match Frame Rate switches the TV to the film's native rate to kill judder.
Tunneled playback — hardware A/V sync for high-bitrate 4K/HDR.
Auto-play next episode with an Up Next card near the end: OK plays now, Back cancels.
External player handoff to VLC, MX or Just Player — a foreground service keeps the engine alive while they play.

Live stats overlay

Codec, content fps vs display Hz, dropped frames, live stream bitrate and how much RAM runway the engine holds — refreshed every second.

Analyze media specs

An on-device probe reads each file's real codec, resolution, HDR type and audio — down to Dolby Vision profile strings (dvhe.07.06), HDR10 and TrueHD Atmos.

Display control

Pin the whole app UI to a chosen refresh rate, or leave it to the system.

Sharing

A media server for Kodi

One toggle publishes the whole library — movies and TV with covers and titles — as a UPnP/DLNA media server. Kodi or any UPnP client browses Beam and direct-plays from Mega, your seedbox and local drives, with Beam's engines relaying the original bytes. Survives reboots until you switch it off.

Engine

Under the hood

Every source looks identical to the decoder, profiled on real hardware until cold starts, seeks and steady-state all held up.

Embedded engine

Runs as a local server; the player streams from 127.0.0.1, so cloud, seedbox and USB all look identical to the decoder.

Tuned RAM read-ahead

A 128 MB player buffer over hundreds of MB of engine read-ahead, profiled until a memory-constrained 2015 SHIELD held smooth.

Dual engines

A tuned stable engine by default, with a one-toggle experimental engine for A/B testing on your own device.

Room-backed catalog

Library, watch state and specs in a relational store; the app starts from cache instantly and scans refresh incrementally.
